# 💸 ExpenseMate  
### Expense Management Application  

ExpenseMate is a simple and intuitive expense management application developed as part of an **App Development Lab project**. It helps users track daily expenses, categorize spending, and manage personal finances efficiently.

---

## 📌 Overview  
ExpenseMate provides a lightweight solution to record and organize financial transactions. The application allows users to maintain a clear view of their spending habits and supports better financial decision-making.

---

## 🚀 Features  
- Add new expenses with relevant details  
- Edit and update existing expense records  
- Delete unwanted entries  
- Categorize expenses (Food, Travel, Bills, etc.)  
- View complete transaction history  
- Simple and user-friendly interface  

---

## 🛠️ Tech Stack  
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL (managed using phpMyAdmin)

## ▶️ How to Run  

1. Clone the repository  
   ```bash```
   git clone https://github.com/your-username/ExpenseMate.git
2. Move the project to your server directory
   For XAMPP → htdocs
   For WAMP → www
3. Start Apache and MySQL from your server
4. Import the database
   Open phpMyAdmin
   Create a new database (e.g., expensemate)
   Import the provided .sql file
5. Configure database connection
   Update database credentials in your config file
6. Run the application
   Open browser and go to: http://localhost/ExpenseManager

The goal of this project is to demonstrate fundamental app development concepts such as CRUD operations, database integration, and user interface design through a practical expense tracking application.
